Example 4

The following "story" (div with paragraphs and a list) is editable. It is configured with a custom 'onSave' function 

$('.demo4').quickEdit({
    submitOnBlur: true,
    html: true,
    onSave: function (t) {
        $('#specialnote').css('display', 'none')
        // do something clever with the updated text
        $.post('api.php', function(resp){
            // updated!
        })
    },
    onEditing: function (t) {
        // the editor has been activated for this story, so
        // we will show a special note. 
        $('#specialnote').css('display', 'block')
    }
}
